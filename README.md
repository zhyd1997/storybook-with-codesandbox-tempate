## Implementation

```shell
### Create React App
### you can use other frameworks that storybook supports!
npx create-react-app my-app --template typescript

### Setup Storybook

### you can specify any version of storybook
cd my-app && npx storybook init

### Start Storybook
npm run storybook
```
